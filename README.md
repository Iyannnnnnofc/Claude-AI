# Claude-AI
AI Claude berbasis Python yang berjalan di terminal / Termux Android.

---

## Penjelasan
Chatbot teks sederhana menggunakan API Claude. Tanya apa saja, AI langsung menjawab. Tidak butuh library tambahan, cukup Python bawaan.

---

## Cara Run
```
python ai_chat.py
```
Ketik pertanyaan lalu Enter. Ketik `exit` untuk keluar.

---

## API
- https://api.azbry.com/api/ai/claude?q={pertanyaan}
